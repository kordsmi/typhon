# Проект Typhon

## Общее описание
Typhon - это транспилятор с языка Python на JavaScript.

Проект находится в процессе разработки.
Главная цель - транспиляция кода, учитывающая особенносит языков python и JavaScript.

Основные особенности (предполагаемые) - анализатор кода с возможностью расширения функционала транспилятора 
средствами python

## Структура проекта

### Основные файлы
- `typhon.py` - точка входа в приложение
- `README.md` - описание проекта и его целей
- `requirements.txt` - зависимости проекта
- `runtime/` - каталог с runtime-файлами
- `tests/` - каталог с тестами
- `typhon/` - основной каталог с исходным кодом

### Зависимости
- pytest - для запуска тестов
